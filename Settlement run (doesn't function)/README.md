Settlement UI - trail

Static prototype of a settlement page for a checkout / warehouse flow.

Files:
- index.html - main page
- css/styles.css - styling
- js/app.js - client-side logic (add items, calculate totals, preview, export CSV)

How to open:
1. Open `index.html` in your browser (double-click or use Live Server extension).
